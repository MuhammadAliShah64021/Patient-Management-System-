# 🏥 Patient Management System

A console-based **Patient Management System** developed in **C++** that allows users to efficiently manage patient records using a simple menu-driven interface. The project demonstrates the use of **Object-Oriented Programming concepts, file handling, arrays, and basic data management techniques**.

---

## 📖 Overview

The Patient Management System is designed to help users maintain patient records through a simple command-line interface. All patient information is stored in a text file, allowing the data to persist even after the application is closed.

---

## ✨ Features

- ➕ Add new patient records
- 📋 View all registered patients
- 🔍 Search patients by ID
- ❌ Delete patient records
- 💾 Automatic file saving
- 📂 Load existing patient records on startup
- 📊 Display available patient slots
- 🖥️ Interactive menu-driven interface

---

## 🛠️ Built With

- **Language:** C++
- **Concepts:** Object-Oriented Programming (OOP)
- **File Handling:** `ifstream` / `ofstream`
- **IDE:** Visual Studio Code / Dev-C++ / Code::Blocks
